Changit
---

An over-engineered tool to change git config for multiple projects at once.

## Installation

```ruby
gem install changit
```

## Usage

In a hub-directory (that contains multiple git projects), add a `.gitconfig`
file with your configuration, and run:

```sh
changit
```
